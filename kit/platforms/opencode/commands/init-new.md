Use the `coding-agent-setup-project` skill to start a new project.

Requirements:
- Use the Plan agent — don't create any files until there's a clear plan and confirmation
- Ask one question at a time, not all at once
- After confirmation, scaffold and set up the project
- Once the project is created, run the `coding-agent-scan-project` skill to finish the setup
